# متجر إلكتروني احترافي

متجر إلكتروني باللغة العربية مبني بالكامل على HTML, CSS, و JavaScript — بدون حاجة لخادم (Backendless).

## الميزات

- أقسام متعددة (إلكترونيات، ملابس، أثاث، إلخ)
- تحميل المنتجات من Google Apps Script
- تحميل الولايات الجزائرية من مصدر خارجي
- سعر توصيل ديناميكي حسب الولاية
- إرسال الطلبات مباشرة إلى Google Forms
- تصميم سريع، متجاوب، ويدعم اللغة العربية (RTL)

## كيفية الاستخدام

1. شغّل المتجر مباشرة عبر GitHub Pages.
2. أضف منتجاتك عبر Google Sheet + Apps Script.
3. عدّل أسعار التوصيل في ملف `data/delivery_fees.json`.
